# Estudo de Caso 1 — Sistema de Pet Shop

## Situação

Um pet shop deseja criar um sistema simples para cadastrar animais.

Cada animal deverá possuir:

- nome
- tipo
- idade

O sistema deverá permitir:

- cadastrar animais
- mostrar animais cadastrados

---

# Objetivo do Exercício

Os alunos deverão praticar:

- criação de classes
- criação de objetos
- atributos
- métodos
- composição de objetos
- listas dentro de classes

---

# Regras do Exercício

- Utilizar **somente Programação Orientada a Objetos**
- Não utilizar programação procedural
- Todas as ações devem acontecer através de objetos e métodos

---

# Estrutura Esperada

## Classe Animal

A classe deverá possuir:

### Atributos

- nome
- tipo
- idade

### Métodos

- mostrar_dados()

---

## Classe PetShop

A classe deverá possuir:

### Atributos

- lista_animais

### Métodos

- cadastrar_animal()
- mostrar_animais()

---

# Desafio

Crie um sistema onde:

1. O usuário possa cadastrar vários animais
2. Os animais sejam armazenados dentro do PetShop
3. O sistema mostre todos os animais cadastrados

---

# Exemplo de Saída Esperada

```text
Nome: Rex
Tipo: Cachorro
Idade: 5 anos

Nome: Mimi
Tipo: Gato
Idade: 2 anos
```
