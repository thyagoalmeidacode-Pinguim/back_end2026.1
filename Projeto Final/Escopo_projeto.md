# Escopo do Projeto Web em Django para
## Objetivo:

Desenvolver um projeto web utilizando o framework Django, proporcionando aos alunos uma introdução prática aos conceitos fundamentais de programação web e ao desenvolvimento de aplicações web dinâmicas.

## Tecnologias:

1. Django: Framework Python para desenvolvimento web.
2. Python: Linguagem de programação.
3. HTML: Linguagem de marcação de hipertexto.
4. CSS: Linguagem de estilo para definir a aparência das páginas web.
5. Bootstrap: Framework CSS para design responsivo e desenvolvimento front-end.
6. Banco de dados: SQLite (por padrão no Django), mas pode ser substituído por outros como PostgreSQL ou MySQL.

## Conceitos a serem abordados:

1. CRUD: Criação, Leitura, Atualização e Deleção de dados.
2. Modelos: Representação das estruturas de dados do aplicativo.
3. Views: Lógica de processamento das requisições HTTP.
4. Templates: Criação das interfaces visuais das páginas.
5. Context: Passagem de dados dos views para os templates.
6. Sessões: Manutenção de estado entre as requisições.
7. Herança de templates: Reutilização de blocos de HTML em diferentes templates.
8. URLs: Mapeamento de URLs para as views.
9. Forms: Criação de formulários para entrada de dados.

## Exemplo de aplicação:

### BLOG SIMPLES, onde os usuários podem:

1. Cadastro de usuários.
2. Login e logout.
3. Criação de posts.
4. Edição e exclusão apenas dos próprios posts.
5. Visualização da lista de posts.
6. Visualização dos detalhes de um post.
7. Cadastro de comentários em um post.
8. Busca de posts por título ou conteúdo.

### GERENCIAMENTO DE TAREFAS, onde os usuários podem:

1. Cadastro de usuários.
2. Login e logout.
3. Criação de tarefas.
4. Edição e exclusão apenas das próprias tarefas.
5. Listagem de tarefas do usuário.
6. Alteração do status da tarefa (Pendente, Em andamento ou Concluída).
7. Busca de tarefas por título ou descrição.

### AGENDA DE EVENTOS, onde os usuários podem:

1. Cadastro de usuários.
2. Login e logout.
3. Criação de eventos.
4. Edição e exclusão apenas dos próprios eventos.
5. Listagem de eventos.
6. Visualização dos detalhes de um evento.
7. Busca de eventos por título ou local.


### BIBLIOTECA, onde os usuários podem:
1. Cadastro de usuários.
2. Login e logout.
3. Cadastro de livros.
4. Edição e exclusão apenas dos livros cadastrados pelo usuário.
5. Listagem de livros.
6. Registro de empréstimos e devoluções.
7. Busca de livros por título, autor ou categoria.

### PORTAL DE NOTÍCIAS, onde os usuários podem:
1. Cadastro de usuários.
2. Login e logout.
3. Publicação de notícias.
5. Edição e exclusão apenas das próprias notícias.
6. Listagem de notícias.
7. Visualização dos detalhes de uma notícia.
8. Busca de notícias por título, categoria ou conteúdo.
