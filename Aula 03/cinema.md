# Estudo de Caso  — Sistema de Cinema

## Situação

Um pequeno cinema deseja criar um sistema simples para controlar a venda de ingressos.

O sistema deverá:

- Cadastrar filmes
- Mostrar filmes em cartaz
- Registrar vendas de ingressos
- Calcular valor arrecadado
- Mostrar quantidade de ingressos vendidos

---

# Objetivo do Projeto

Os alunos deverão desenvolver funções para:

- Cadastrar filme
- Mostrar filmes
- Vender ingresso
- Calcular arrecadação
- Mostrar total de ingressos vendidos

---

# Regras

- Utilizar listas
- Utilizar funções
- Utilizar parâmetros
- Utilizar `return`
- Utilizar repetição (`for` e `while`)
- Organizar o sistema com menu

---

# Contextualização com Vida Real

Quando você compra ingresso em um cinema:

- o sistema mostra os filmes
- registra sua compra
- calcula o valor
- guarda as vendas

Tudo isso acontece através de funções e armazenamento de dados.
