# Estudo de Caso  — Sistema de Biblioteca

## Situação

Uma biblioteca escolar deseja um sistema simples para controlar empréstimos de livros.

O sistema deverá:

- Cadastrar livros
- Mostrar livros disponíveis
- Registrar empréstimos
- Mostrar quantidade de livros emprestados

---

# Objetivo do Projeto

Os alunos deverão desenvolver funções para:

- Cadastrar livro
- Mostrar livros
- Realizar empréstimo
- Contar empréstimos

---

# Regras

- Utilizar listas
- Utilizar funções
- Utilizar parâmetros
- Utilizar `return`
- Utilizar repetição
