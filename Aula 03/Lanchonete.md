# Estudo de Caso — Sistema de Lanchonete

## Situação

Uma pequena lanchonete precisa de um sistema simples para organizar os pedidos dos clientes.

O sistema deverá:

- Cadastrar lanches
- Mostrar o cardápio
- Registrar pedidos
- Calcular o valor total da compra

---

# Objetivo do Projeto

Os alunos deverão criar funções para:

- Cadastrar lanche
- Mostrar cardápio
- Registrar pedido
- Calcular total da compra

---

# Regras

- Utilizar listas
- Utilizar funções
- Utilizar estrutura de repetição
- Utilizar parâmetros
- Utilizar `return`
- Mostrar mensagens organizadas

#Reflexão

Aplicativos como:

- iFood

- cadastrar produtos
